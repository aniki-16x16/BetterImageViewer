# Better Image Viewer

<div align="center">
  <p><strong>一款专为 Windows 平台打造的轻量级、高性能图片查看器，旨在完美替代 Windows 自带的“照片”应用。</strong></p>
</div>

本项目基于 **Rust** 和 **egui** 框架开发，专注于提供极致的响应速度和流畅的交互体验，让图片浏览回归简单与纯粹。

## ✨ 核心特性

- 🚀 **极速冷启动**：得益于 Rust 的底层性能优化，软件的冷启动速度远超 Windows 自带的图片查看器，即点即开，告别等待。
- 🌊 **高刷屏丝滑体验**：针对高刷新率显示器进行了深度优化，无论是缩放、拖拽还是切换图片，都能保证如丝般顺滑的视觉体验。
- 🖼️ **多图浏览优化**：重新设计了多图片查看的交互逻辑，在同一文件夹下切换图片更加自然流畅，大幅提升连续浏览的效率。
- 🦀 **极致性能与低占用**：基于 `egui` 即时模式 GUI 框架构建，不仅界面简洁现代，而且内存占用极低，运行轻快。

## 🎯 为什么开发这个项目？

Windows 自带的图片查看器在经历了多次迭代后，虽然功能越来越丰富，但也变得越来越臃肿。在日常使用中，我们往往只需要一个能够**快速打开**、**流畅缩放**的纯粹工具。特别是在高刷新率屏幕普及的今天，自带应用的卡顿感和较慢的冷启动速度显得尤为明显。

Better Image Viewer 就是为了解决这些痛点而生，它剥离了冗余功能，将所有的性能都倾注于“查看图片”这一核心需求上。

## 🛠️ 安装与构建

确保你的系统已经安装了 [Rust 工具链](https://www.rust-lang.org/tools/install)。

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/yourusername/better_image_viewer.git
   cd better_image_viewer
   ```

2. 编译并运行项目：
   ```bash
   cargo run --release
   ```
